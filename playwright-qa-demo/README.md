# Playwright QA Demo

Projeto MVP de automação de testes com Playwright para estudos de QA moderno.

## Tecnologias

- Playwright
- JavaScript
- Node.js
- GitHub Actions
- Dotenv
- ESLint
- Prettier

## Estrutura

```text
playwright-qa-demo/
├── tests/
│   ├── e2e/
│   ├── api/
│   └── fixtures/
├── pages/
├── utils/
├── data/
├── docs/
├── reports/
├── .github/workflows/
├── playwright.config.js
├── package.json
└── README.md
```
